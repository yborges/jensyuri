Question 4

I'm more familiar with Java. The syntax of JS and Java are similar and based on C. Java is pre compiled and therefore it achieves better performance. Python and JS are interpreted at runtime and are therefore slower. JS relies much more on callback functions. Java is statically typed whereas Python and JS are dynamically typed, which leads to more freedom when coding, but that comes with a price: unstructured code.

Question 5

AJAX means Asynchronous JavaScript And XML. It is asynchronous because the browser(client) is still able to perform actions while waiting for a request from the server. It can also be set to synchronous which would lock the browser until the server responds.

Question 6

The role of the host objects in JavaScript is to let JS change their properties. e.g. the document and window host objects are accesible via JS and let us change properties of html elements and access the size of the host window. There are no equivalents in Java or Python because these programs are to some extent executed in the same environment as far as the programs are concerned. Although the host computers aren't the same and their properties can actually be accessed but they are not considered objects, most probably because they are read-only.

Question 8

Events are as the name says "things" that happen to elements in the DOM. When an event is triggered, event listeners can react by executing an action. Event listeners are also elements in the DOM that have event handlers attached to them.
