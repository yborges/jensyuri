Chris Anderson wrote The Long Tail, which first appeared in Wired in October 2004. The theory of the long tail says that our culture and economy is increasingly shifting away from a focus on a relatively small number of "hits" that are the most popular (at the head of the demand curve) to a relatively big number of the less popular niches in the tail. He states that the less popular niches can return as much profit as the sales of the popular items. It may be hard for physical stores to have such large inventory, due to space limitations and storage costs. But for the online stores, it will be a lot easier to handle this broad variety of items that customers want because, online stores can have decentralized storages and may be able to buy items from retailers when the don't have it in stock.

The Common Log Format is a standardized text file format used by web servers when generating server log files.

The Common Log Format contains the folowing information (found on the apache website):

  • 1. The IP address of the client (remote host) which made the request to the server.
  • 2. The user id of the person requesting the document as determined by HTTP authentication.
  • 3. The time that the server finished processing the request.
  • 4. The request line.
  • 5. The status code that is returned to the client by the server.
  • 8. The size of the object returned to the client.

This can be used by web log analysis software to identify the user, the way that the user requested and the time and timezone the user is in.
