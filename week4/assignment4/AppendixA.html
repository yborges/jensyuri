1.3.1 Info and Relationships

This guideline has been fulfilled when it comes to semantics by using the tag strong in stead of bold. We don't have content for which the sequence is important, except for the headline. Due to the position of the headline in the html code it is guaranteed to be placed in the right position by all browsers. This guideline is thereby met.
